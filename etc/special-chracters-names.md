# 특수문자 이름들

| 기호 | 이름 (한국어)      | 이름 (영어)                  |
| ---- | ------------------ | ---------------------------- |
| ~    | 틸드               | Tilde                        |
| `    | 백틱               | Backtick, Grave accent       |
| !    | 느낌표             | Exclamation mark             |
| @    | 앳                 | At sign                      |
| #    | 샵, 해시           | Hash, Number sign, Pound     |
| $    | 달러               | Dollar sign                  |
| %    | 퍼센트             | Percent sign                 |
| ^    | 캐럿               | Caret                        |
| &    | 앰퍼샌드           | Ampersand                    |
| \*   | 애스터리스크, 별표 | Asterisk                     |
| ( )  | 소괄호             | Parentheses, Round brackets  |
| -    | 하이픈, 대시       | Hyphen, Dash                 |
| \_   | 언더스코어         | Underscore                   |
| =    | 이퀄 사인          | Equals sign                  |
| +    | 플러스             | Plus sign                    |
| { }  | 중괄호             | Curly braces, Curly brackets |
| [ ]  | 대괄호             | Square brackets              |
|  |   | 파이프             | Pipe, Vertical bar           |
| \    | 백슬래시           | Backslash                    |
| /    | 슬래시             | Slash, Forward slash         |
| :    | 콜론               | Colon                        |
| ;    | 세미콜론           | Semicolon                    |
| "    | 큰따옴표           | Double quote                 |
| '    | 작은따옴표         | Single quote, Apostrophe     |
| < >  | 꺾쇠 괄호          | Angle brackets               |
| ?    | 물음표             | Question mark                |
| ,    | 콤마               | Comma                        |
| .    | 점, 닷             | Period, Dot, Full stop       |
